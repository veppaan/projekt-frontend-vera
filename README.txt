Webbplatsen är gjord av Vera Kippel för projektarbete i frontend-baserad webbutveckling 2025
Webbplatsen "VattenTempen" innehåller tre api:er som kommunicerar med 
varandra för att visa data om havet som användaren väljer.

Parcel har använts för att optimera projektets filer.
Länk till publicerad webbapplikation : https://vattentempen.netlify.app
